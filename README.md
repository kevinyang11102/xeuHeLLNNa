# 智慧图书管理系统简介 java1194

## 项目概述

智慧图书管理系统是一款基于前后端分离架构的图书管理软件。系统采用Spring Boot和Vue技术栈，致力于提供高效、稳定的图书管理服务。

## 技术栈

- 后端：Spring Boot框架，Java 1.8
- 前端：Vue2
- 数据库操作：MyBatis-Plus
- 数据库：MySQL

## 功能模块

### 系统角色

#### 管理员

- 图书信息管理：添加、编辑、删除
- 库存管理
- 用户信息管理：注册审核、权限分配、查看借阅记录

#### 用户

- 图书搜索与查看
- 借阅、续借、归还操作
- 个人借阅历史与逾期情况查询

## 运行环境

- 后端：Java 1.8及以上版本环境
- 前端：现代浏览器
- 数据库：MySQL 5.7及以上版本
